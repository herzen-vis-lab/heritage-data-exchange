# Heritage Data Exchange

Прототип федеративного обмена объектами цифрового культурного наследия
между организациями (университеты, музеи, библиотеки).

Разработан в рамках исследования РГПУ им. А. И. Герцена.

## Что реализовано

- Объектное хранилище цифровых объектов наследия
- Карточка объекта с атрибутированием
- API: выдача карточек с фильтром по группам объектов

- Веб-интерфейс для администраторов и экспертов

## Что в планах

- API: приём карточки от внешней организации
- Механизм дедупликации между узлами
- Взаимное обогащение атрибутов при совпадении объектов
- Мультиузловой обмен между двумя живыми экземплярами

## Структура репозитория (разрабатывается)

/docs — архитектурные схемы и описание протокола обмена

/schema — JSON-схема карточки объекта

/api — OpenAPI спецификация двух ручек обмена
