# Heritage Data Exchange

Прототип федеративного обмена объектами цифрового культурного наследия между университетами, музеями и библиотеками.

Проект исследует децентрализованный обмен карточками объектов без единого централизованного каталога.

Разрабатывается в РГПУ им. А. И. Герцена.

## Идея
Каждая организация:
* хранит данные локально
* публикует карточки объектов
* обменивается метаданными с другими узлами
* участвует во взаимном обогащении атрибутов

## Проблема
Существующие форматы (MARC21, Dublin Core, IIIF, EDM) плохо подходят для федеративного обмена между независимыми организациями:
* отсутствует идентификация узлов
* архитектура ориентирована на централизованные системы

Проект исследует формат обмена для децентрализованной сети узлов.
